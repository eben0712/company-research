# company-research：上市公司深度覆盖分析

> 某券商行业研究实习生被愚钝的 AI 折磨崩溃后，一怒之下写了这个 skill。

输入公司名或股票代码，自动搜索、分析、输出一份完整研报——商业模式、护城河、竞争格局、财务趋势、盈利预测、风险提示全部覆盖，最后导出 Markdown 和 Word 两份文件。

## 安装

将 `company-research/` 目录放到你的 Claude Code skills 目录下：

```
~/.claude/skills/company-research/
```

重启 Claude Code 即可。触发方式：提到"分析XX公司""深度研究XX""XX的商业模式是什么"等关键词即自动加载。

## 使用流程

1. 触发后弹出三个选择题（分析深度/关注重点/准确度偏好），选完直接开始
2. 三轮搜索——基本面+行业→财务细节+竞品对比→风险+交叉验证
3. 按 11 模块框架输出完整报告（Markdown）
4. 审阅后一句命令导出 Word

## 支持的公司类型

不只是消费品，五种类型自动识别并调整分析权重：

| 类型 | 代表公司 | 核心矛盾 |
|------|----------|----------|
| 渠道型 | 锅圈、蜜雪冰城 | 加盟商盈利 vs 门店密度 |
| 产品型 | 有友、卫龙 | 品类天花板 vs 品牌心智 |
| 制造型 | 海天、安井 | 规模效应 vs 产能利用率 |
| 平台型 | 美团、拼多多 | 网络效应 vs take rate |
| 内容/IP型 | 完美世界、阅文 | IP生命周期 vs 爆款持续性 |

## 导出效果

Word 文档：黑体标题/宋体正文、深蓝表头白字+交替行底色、页码页脚。

## 局限

- 数据质量取决于 WebSearch 搜索结果
- 盈利预测是方向性参考，非精确预测
- 适合覆盖式研究，不能替代深入的一手调研
