# 搜索关键词策略库

## 使用方式

根据分析模块 + 公司所处行业，选择对应关键词组合执行 WebSearch。
关键词中"公司名"替换为目标公司全称或简称，"股票代码"替换为实际代码，"XX行业"替换为具体行业名。

**重要**：下文"通用关键词"适用于消费品/制造/渠道类公司。对于游戏、科技、医药、金融等行业的公司，必须同时参考"行业适配关键词"章节替换对应的行业术语，否则搜索结果会严重偏离。

## 搜索优先级

### 第一轮（并行5-8个）

**通用**（所有行业必搜）：
1. "公司名 发展历程 创始人 商业模式" → 模块1
2. "公司名 股票代码 年报 营收 净利润 毛利率" → 模块1/7
3. "XX行业 市场规模 增速 竞争格局" → 模块3
4. "公司名 竞争对手 对比 市场份额" → 模块4
5. "公司名 股权结构 治理 分红" → 模块1

**按行业替换**（从行业适配关键词中选2-3个核心项）：
6-8. 根据公司所处行业，从上方"行业适配关键词"中选取最核心的2-3个搜索

### 第二轮（并行3-5个）

**通用**：
1. "公司名 财务 应收账款 现金流 ROE 费用率" → 模块7
2. "公司名 vs 主要竞品 深度对比" → 模块4
3. "国际对标 XX模式 类似公司 对比" → 模块5

**按行业替换**：
4-5. 从行业适配关键词中选剩余项做精确搜索（各渠道/各产品线/各区域拆分数据）

### 第三轮（2-3个）

1. "公司名 风险 减持 关停 纠纷 投诉" → 模块10
2. "XX行业 最新动态 趋势" → 模块3交叉验证
3. 任何第一/二轮中数据存疑的补充搜索

## 一手资料获取

- A股年报/招股书：WebSearch "公司名 股票代码 巨潮资讯网 年报 PDF" 或 "公司名 东方财富 财务数据"
- 港股：WebSearch "公司名 港交所 年报"
- 用WebSearch获取关键数字后，准确模式下用Read工具直接读PDF原文关键章节
