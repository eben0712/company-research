# 公司类型判定与模块权重指引

## 判定方法

在搜索第一轮信息后，根据以下特征判定公司类型，自动调整11个模块的分析深度权重。

## 五种公司类型

### 1. 渠道型公司

**特征识别**：收入主要来自向加盟商/经销商供货；门店网络是核心资产；自有品牌占比高但多为ODM贴牌；供应链效率是竞争关键。

**代表公司**：锅圈、蜜雪冰城、绝味食品、紫燕百味鸡、杨国福、名创优品

**核心矛盾**：公司与加盟商的利益博弈——公司的增长依赖加盟商扩张，但加盟商盈利空间被门店加密和低价策略压缩。

**模块权重调整**：
- ★★★ 模块2（三层拆解）：重点穿透"真正客户是加盟商"这一层
- ★★★ 模块5（竞品生态位）：重点在渠道密度和下沉能力的横向对比
- ★★★ 模块6（渠道与产品矩阵）：重点在加盟模式、门店模型、区域保护
- ★★☆ 模块4（护城河）：重点区分"规模壁垒"和"虚假繁荣"
- ★☆☆ 模块3（行业分析）：简化为赛道空间判断
- ★★★ 模块10（风险）：重点在加盟商盈利、闭店率、食品安全

### 2. 产品型公司

**特征识别**：核心品类≤3个且占收入80%+；品牌=品类的消费心智强；自产为主或ODM比例高但品类聚焦；渠道依赖外部（经销商/商超/平台）。

**代表公司**：有友食品、卫龙、涪陵榨菜、老干妈、王小卤

**核心矛盾**：品类天花板——一只产品能卖多大？能否跨品类复制成功？

**模块权重调整**：
- ★★★ 模块2（三层拆解）：重点穿透"品牌溢价 vs 品类红利"
- ★★★ 模块4（护城河）：重点在品牌心智=品类的证据、复购率、工艺壁垒
- ★★★ 模块5（竞品生态位）：重点在生态位分化和跨品类竞争对手
- ★★☆ 模块7（新业务）：重点在第二曲线可行性和历史孵化能力
- ★★★ 模块10（风险）：重点在品类老化、渠道依赖、原材料

### 3. 制造型公司

**特征识别**：产能规模和利用率是关键变量；成本优势是核心壁垒；下游客户多元化；多品类多SKU矩阵。

**代表公司**：海天味业、安井食品、伊利股份、金龙鱼、三全食品

**核心矛盾**：规模和效率——能否持续压低单位成本？产能扩张是否被需求消化？

**模块权重调整**：
- ★★★ 模块2（三层拆解）：重点在自产 vs 外购、产能利用率、单品成本结构
- ★★★ 模块3（行业分析）：重点在行业集中度和规模效应曲线
- ★★☆ 模块4（护城河）：重点在规模壁垒和成本优势的可维持性
- ★★☆ 模块8（财务趋势）：重点在毛利率、产能利用率、资本开支
- ★★★ 模块10（风险）：重点在原材料价格、产能过剩、替代品

### 4. 平台型公司

**特征识别**：双边/多边市场；收入来自交易抽成或广告；网络效应是核心增长飞轮；轻资产重运营。

**代表公司**：美团、拼多多、滴滴、贝壳、抖音电商

**核心矛盾**：网络效应的深度——用户多了商家才来，商家多了用户才来，这个正循环是否足够强？

**模块权重调整**：
- ★★★ 模块2（三层拆解）：重点在双边生态和take rate
- ★★☆ 模块4（护城河）：重点在网络效应和转换成本
- ★★★ 模块5（竞品生态位）：重点在生态位重叠度和补贴战风险
- ★☆☆ 模块7（新业务）：平台型公司新业务通常很多，重点在协同逻辑
- ★★★ 模块10（风险）：重点在监管、垄断、流量成本

### 5. 内容/IP型公司

**特征识别**：核心资产为知识产权（游戏IP、小说版权、影视内容、专利技术）；
收入高度依赖少数爆款产品；产品有明确生命周期（上线→爬坡→成熟→衰退）；
研发/创作投入大但边际复制成本低；用户为内容付费而非实体商品。

**代表公司**：完美世界、米哈游、阅文集团、光线传媒、网易（游戏部分）、腾讯音乐

**核心矛盾**：爆款持续性——能否持续产生下一个爆款？IP 能否跨形态反复变现？
同时面临 IP 老化（用户代际更替）、研发人才流失、单产品集中度风险。

**模块权重调整**：
- ★★★ 模块2（三层拆解）：重点在"IP的跨形态复利变现"——一款IP如何通过端游/手游/影视/衍生品反复赚钱
- ★★★ 模块4（护城河）：重点在IP储备厚度、自研能力、用户粘性、品牌/IP老化风险
- ★★★ 模块7（新业务/第二曲线）：重点在pipeline厚度、新品类成功率、跨品类复制能力
- ★★☆ 模块6（渠道与产品矩阵）：适配为"产品生命周期矩阵 + 各平台/各区域收入拆解"
- ★★★ 模块10（风险）：重点在单品集中度、IP老化、核心人才流失、版号/审查合规

## 权重符号说明

- ★★★ = 深度分析（本文最核心的2-3个模块）
- ★★☆ = 标准分析（按模板正常覆盖）
- ★☆☆ = 简化处理（缩短篇幅，只抓关键判断）
